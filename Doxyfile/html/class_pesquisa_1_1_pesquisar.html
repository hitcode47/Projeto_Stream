bool Pesquisa::Pesquisar::confirma_Busca ( )

Confirma a busca realzada pelo usuário.

Acessa os _resultados definidos pela busca e retorna no máximo as as três músicas com as sintaxes parecidas, dando o poder de escolha para o usuário.

Retorna
Verdadeiro se ocorreu com sucesso, Falso caso contrário.
