void Musica::adicionar_musica ( const std::string &  nomeArquivo)

Adiciona uma música ao sitema.

É solicitado ao usuário o nome do artista, bem como o link da música a ser inserida. É realizado o tratamento das palavras nos nomes e, por fim, o arquivo "Musica_Exemplo.txt" é atualizado.

void Musica::Remove_musica ( const std::string &  nomeArquivo)

Remove uma música do arquivo.

É solicitado ao usuário o nome do artista e da música a ser removida e assim uma busca no arquivo "Musica_Exemplo.txt" é iniciada. Caso encontrada, a música é excluída e o arquivo atualizado.
